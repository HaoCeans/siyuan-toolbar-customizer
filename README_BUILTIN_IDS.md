## 💡 使用技巧

### 基础用法
在"思源内置功能"类型的按钮配置中，直接填写 ID：
```
toolbarFile
```

### 配合模拟点击序列
可以先打开某个面板，再执行其他操作：
```
toolbarFile        ← 打开文档树
text:我的文档      ← 点击名为"我的文档"的项目
```

### 多步骤操作
```
barSettings        ← 打开设置
text:外观          ← 点击"外观"标签
text:明亮           ← 选择明亮主题
```

## 🔍 如何找到其他功能的 ID？

### 方法 1：使用浏览器开发者工具
1. 按 `F12` 打开开发者工具
2. 点击"选择元素"工具（`Ctrl+Shift+C`）
3. 点击你想要的按钮
4. 在 Elements 面板查看元素的 `id` 或 `data-id` 属性

### 方法 2：查看思源源码
访问思源笔记的 GitHub 仓库查看源码中的按钮定义

### 方法 3：尝试常见模式
- 顶部工具栏：`bar` + 功能名（如 `barSearch`）
- 菜单项：`menu` + 功能名（如 `menuSync`）
- 移动端工具栏：`toolbar` + 功能名（如 `toolbarFile`）

## ⚠️ 注意事项

1. **平台差异**：电脑端和手机端的某些功能 ID 不同
   - 电脑端：`barFile`
   - 手机端：`toolbarFile`

2. **版本差异**：不同版本的思源笔记可能有不同的 ID，建议实际测试

3. **备选方案**：如果 ID 查找失败，可以使用"模拟点击序列"配合 `text:` 前缀

4. **SVG 元素**：某些按钮是 SVG 元素，插件已自动处理，可以正常点击
